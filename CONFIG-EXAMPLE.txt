VmShellPAY-HKD for WHMCS 8
Configuration Example

1. VmShell PAY AppId:
   your_app_id

2. VmShell PAY AppSecret:
   your_app_secret

3. 联系邮箱:
   support@example.com

4. 平台争议中心链接:
   https://vmshell.win/disputes/{dispute_id}

5. 支付方式:
   - 支付宝.中国: Yes
   - 支付宝.香港: Yes
   - 微信: Yes

6. 默认支付方式:
   alipay_cn

7. 收款货币种类:
   USD

8. 汇率来源:
   api

9. 自动汇率接口:
   https://api.frankfurter.dev/v2/rate/{from}/{to}

10. 手工汇率表:
   USD=7.80
   EUR=8.45
   GBP=9.95
   SGD=5.78
   JPY=0.053

11. 收款手续费比例:
   2.9

12. 退款手续费每笔:
   2.00

13. VmShellPAY callback URLs:
   Payment notify:
   https://your-whmcs-domain/modules/gateways/callback/vmshellpay_payment_notify_url.php

   Refund notify:
   https://your-whmcs-domain/modules/gateways/callback/vmshellpay_refund_notify_url.php

   Dispute notify:
   https://your-whmcs-domain/modules/gateways/callback/vmshellpay_dispute_notify_url.php

14. Internal defaults used by this version:
   - API base URL: https://vmshell.win
   - Sign type: HMAC-SHA256
   - Terminal: auto
   - Payment scene: auto
   - Order prefix: WHMCS
   - Notify mode: compat
   - Exchange rate timeout: 10 seconds
